# ⏳ TimeBanking Platform for College Students

A full-stack web application that enables college students to exchange skills and services using **time credits instead of money**. The platform encourages peer-to-peer collaboration, skill sharing, and community engagement through a secure and user-friendly interface.

---

## 📌 Project Overview

The TimeBanking Platform is designed to create a collaborative ecosystem where students can offer their skills, request assistance, and earn time credits for every service they provide. These earned credits can later be redeemed to receive help from other students, promoting a non-monetary exchange of knowledge and services.

The project demonstrates full-stack web development concepts using **Flask**, server-side templating, session management, and responsive frontend design.

## 💡 How It Works

1. Users create an account and log in securely.
2. Students offer services such as tutoring, design, programming, or mentoring.
3. Other students browse available services and submit requests.
4. Upon successful completion of a service, the provider earns **time credits**.
5. Earned credits can later be used to request services from other members of the platform.
